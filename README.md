## 🛤️ Personalisation Guide

| What to change | Where |
|---|---|
| Your photo | `<img id="hero-photo" src="...">` in `index.html` |
| GitHub / LinkedIn / Email | All `href` attributes in `index.html` |
| Resume file | `href` on `.btn-resume` anchor |
| Skill percentages | `data-width` + `data-pct` attributes |
| Project descriptions | `.project-desc` paragraphs |
| Typed role titles | `words` array in `script.js` |
| Colour theme | CSS variables at the top of `style.css` |
